# GPT-3 chat demo

## Install

    $ python3 -m venv .venv
    $ source .venv/bin/activate
    $ pip install -r requirements.txt
    $ echo "OPENAI_API_KEY=<insert your OpenAI API key here>" > .env

## Run
    $ python application.py
    => visit http://localhost:5000
